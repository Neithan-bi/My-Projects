📌 Purpose

This dashboard aims to inform, educate, and empower users with accurate and up-to-date COVID-19 data, helping them make safe decisions during the pandemic.

📈 Live Statistics: Access real-time global and local COVID-19 case numbers, recoveries, and fatalities.

🌍 Interactive Maps: Visualize regional impact using map overlays.

📉 Trend Analysis: Identify patterns and projections using interactive graphs.

👥 Who It's For

🧑‍⚕ Healthcare Professionals: Get insights on healthcare loads and regional risks.

🏠 Citizens: Stay informed and take action based on current statistics.

🏛 Policymakers: Use real-time data to support planning and safety measures.
